Paragraph

A paragraph (from the Greek paragraphos, "to write beside" or "written beside") is a self-contained unit of a discourse in writing dealing with a particular point or idea. Paragraphs that are not part of a fiction story are set using web conventions, that is: no indents, with a space between each graf.

HTML

									
										<p>A paragraph (from the Greek paragraphos, "to write beside" or "written beside") is a self-contained unit of a discourse in writing dealing with a particular point or idea. Paragraphs that are not part of a fiction story are set using web conventions, that is: no indents, with a space between each graf.</p>

Story Paragraph

A paragraph (from the Greek paragraphos, "to write beside" or "written beside") is a self-contained unit of a discourse in writing dealing with a particular point or idea. Story paragraphs on Fireside Magazine are indented rather than spaced, as a nod to traditional text block design.

HTML

									
										<div class="story">
<p>A paragraph (from the Greek paragraphos, "to write beside" or "written beside") is a self-contained unit of a discourse in writing dealing with a particular point or idea. Story paragraphs on <em>Fireside Magazine</em> are indented rather than spaced, as a nod to traditional text block design.</p>
</div>

Blockquote

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text, and typically distinguished visually using indentation and a different typeface or smaller size quotation.

HTML

	A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text, and typically distinguished visually using indentation and a different typeface or smaller size quotation.
</blockquote>

    • This is a nested list item
    • This is another nested list item in an unordered list
  • This is the last list item
